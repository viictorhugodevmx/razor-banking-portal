# Razor Banking Portal

Octavo proyecto del programa **Bankaool .NET Fintech Bridge Lab**.

Frontend con Razor Pages para practicar UI bancaria renderizada desde servidor.

## Objetivo

Construir un portal bancario simple usando Razor Pages.

## Stack

- Ubuntu
- .NET 8
- C#
- Razor Pages
- HTML
- CSS

## Conceptos

| JS / Frontend | Razor Pages |
|---|---|
| Pug / HBS / Blade | Razor `.cshtml` |
| Ruta/página | Archivo dentro de `Pages/` |
| Template variables | `@Model.Propiedad` |
| Handler de página | `PageModel` en `.cshtml.cs` |

## Comandos

Restaurar:

```bash
dotnet restore

```

Build:

dotnet build

## Ejecutar:

dotnet run

## Modo watch:

dotnet watch run
